# MovieVerse

A Vite + React movie app demo using TMDB API.

How to add API key:
- Create a `.env` file at project root with:

```
VITE_TMDB_API_KEY=YOUR_API_KEY_HERE
```

Run:

```bash
npm install
npm run dev
```

Features:
- Trending movies (day/week)
- Search with debounce
- Movie details + trailer modal
- Watchlist saved to localStorage
- Light / Dark themes
- English + Georgian (ქართული)
- SCSS styling and Framer Motion animations
